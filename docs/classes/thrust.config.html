Defined in build\doc\src\thrust\config.tsdoc:117

AMD API load

Handles fetching of a current config for the current thrust instance, or the config of the given plugin. Adding the : character requests a specific config plugin. thrust/config!global = thrust!global:config = Thrust instance config from the instance named global.

Parameters:

  • name String

    The name of the instance that is being fetched

  • parentRequire Function

    the require method to be loaded

  • load Function

    Allows the load to inform that AMD for the value to hand off

  • config Object

    The custom configuration.
