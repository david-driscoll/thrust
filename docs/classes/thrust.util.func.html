saveInvoke

(
  • collection
  • method
  • [args]
)

Defined in src\util\function.js:118

Attempts to invoke, similar to _.invoke, but in this case it verifies that the property exist, and also verifies that it is a function, and not the noop method available in thrust.

The intent is a method that allows override of functions, without creating custom code.

Parameters:

  • collection Array | Object

    The container that has the items

  • method String | Function

    The method name on every item, or the method to invoke against each item.

  • [args] Object optional multiple

    The additional arguments to pass onto the method.
