thrustCall

(
  • method
  • facadeAfter
  • args
)
protected

Defined in build\doc\src\thrust\capsule.tsdoc:35

Makes a call to all the modules facades The order of the call depends on the order required. During the startup stage (init, start, ready) facades are called first. During the shutdown state (stop, destroy) facades are called last. This allows modules to startup and shutdown will all the tools it had to begin with.

Parameters:

  • method String

    the method to call

  • facadeAfter Boolean

    calls facade methods before or after module method.

  • args Array

    Args to be passed onto the module method.
